# SQL Assist - Database Management Application

# Project Description
SQL Assist is a Java-based database management application developed using JavaFX and MySQL.
The purpose of this application is to help users manage data in an organized way using categories and items.

User can create categories, add items under categories, update and delete records and view data in a structured format.

# Features
- Create categories 
- Add items under categories 
- Update item details
- Delete items
- View items categories Wise
- Display statistics using chart

# To work this project 
# Requirement
- Java installed in system 
- MySQL installed in system 
- Intellij IDEA Community Edition 

# To run this Application 
1. Open the project
2. Run the main file
3. Enter Database details (Server Location, Database Name, username, password)
4. Then you can use the App

# Database 
There are 3 tables:
- users
- categories
- items

Items are connected to category using category_id.
